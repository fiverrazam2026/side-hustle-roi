# Side Hustle ROI & Burnout Matrix Engine

A responsive, browser-native computation app built to evaluate whether a secondary business venture delivers a healthy financial return on time or risks causing toxic fatigue.

## 🚀 Architectural Features
- **Stress Multiplier Penalty Engine**: Factors in emotional and energetic costs to compute a real take-home hourly value.
- **True Time Asset Auditing**: Integrates administrative and unbilled hours alongside gross monthly business revenues.
- **Performance Optimized Core**: Zero dependency, single file infrastructure designed to run immediately across desktop and mobile browsers.
